## The open-source tax engine for AI

A single binary that runs the full IRS Form 1040 math — locally, offline, inside any AI platform you already use.

Wire it into Claude, ChatGPT, or Gemini as a CLI tool. Tell the AI what's on your tax documents. The engine runs the numbers and hands back a finished return.

```bash
# Create a return
tax return create --year 2025
# → { "returnId": "abc-123" }

# Add your W-2
tax form add --returnId abc-123 --node_type w2 \
  '{"box1_wages": 95000, "box2_fed_withheld": 14200}'

# Compute the full 1040
tax return get --returnId abc-123
```

## Why tax-engine?

1. **Fully local** — runs as a binary on your machine, no data ever leaves, no accounts, no subscriptions
2. **131 input nodes** — covers W-2s, every 1099 variant, all schedules, credits, deductions, and foreign forms
3. **IRS-accurate** — 97+ scenarios sourced directly from IRS publications, validated line-by-line
4. **Works in any AI** — Claude, ChatGPT, Gemini, or any platform that can run a subprocess
5. **MeF XML export** — produces IRS-valid e-file XML ready for a clearinghouse
6. **Open source** — MIT licensed, no black box

## How it works

```
W-2         → wages, withholding    ╮
1099-INT    → interest income       ├→ Form 1040 → Schedule 1 → … → refund / owed
1099-DIV    → dividends             ╯
Schedule C  → self-employment
```

Each tax document is an **input node**. The engine resolves dependencies, runs the IRS math in topological order, and produces every computed line on the 1040.
