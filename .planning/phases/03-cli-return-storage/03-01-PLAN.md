    3. Create engine/cli/store/store.ts with these exports:
    - `nextId(entries: readonly InputEntry[], nodeType: string): string` — counts entries where `e.nodeType === nodeType`, returns `${nodeType}_${String(count + 1).padStart(2, "0")}`. Pure function, no I/O.
    - `async createReturn(year: number, baseDir: string): Promise<{ returnId: string; returnPath: string }>` — generates `crypto.randomUUID()`, creates `${baseDir}/${returnId}/` directory via `ensureDir` from `@std/fs`, writes `meta.json` with `{ returnId, year, createdAt: new Date().toISOString() }`, writes `inputs.json` with `[]`. Returns `{ returnId, returnPath }`.
    - `async loadMeta(returnPath: string): Promise<MetaJson>` — reads `${returnPath}/meta.json`, parses JSON, returns MetaJson. Throws with message `"meta.json not found at ${returnPath}"` if file does not exist (catch Deno.errors.NotFound).
    - `async loadInputs(returnPath: string): Promise<InputEntry[]>` — reads `${returnPath}/inputs.json`, parses JSON, returns InputEntry[]. Throws with message `"inputs.json not found at ${returnPath}"` if file does not exist.
    - `async appendInput(returnPath: string, entry: InputEntry): Promise<void>` — loads existing inputs via `loadInputs`, creates new array `[...existing, entry]`, writes back to `${returnPath}/inputs.json`. Immutable pattern: never mutates the loaded array.

    All functions use `Deno.writeTextFile` and `Deno.readTextFile`. JSON is written with 2-space indent (`JSON.stringify(data, null, 2)`).
